## Instructions

### First, run the backend server:

```bash
python routing/api.py
```

The api runs on the port 8080 by default.

### Then start the frontend app:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
